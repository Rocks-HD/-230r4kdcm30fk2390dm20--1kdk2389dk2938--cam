CRÉDITOS DO APP

EDO nada mais é que a abreviação de "Estamos de Olho". Nossa intenção é a de colocar no circuito político o protagonista desse vínculo, o Cidadão. O intuito é facilitar o processo de transparência e a participação popular visando sempre o bem maior para todos.

Este APP foi desenvolvido para participação do Concurso APP Legislativo 2017 organizado pela Câmara dos Deputados.
